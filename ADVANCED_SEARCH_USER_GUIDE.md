# Advanced Search & Filters - User Guide

## Overview
The Jobs page now has powerful search and filtering capabilities to help users find exactly the right pharmacy job.

---

## Main Features

### 1. Search Bar
**What it does**: Search for jobs by keyword, position, or title

**How to use**:
1. Click on the search bar at the top of the filter panel
2. Type your search query (e.g., "Pharmacist", "Hospital")
3. Results update automatically
4. Clear with the X button or delete the text

**Example searches**:
- `Pharmacist` - Find all pharmacist positions
- `Harare` - Find jobs in Harare
- `Community pharmacy` - Find community pharmacy jobs

- And more...

**How to use**:
1. Click "Position/Role" header to expand

- Bulawayo

- Mutare
- Gweru

- And more...

#### Employment Type Filter
**Options available**:
- Full-Time
- Part-Time
- Contract
- Internship
- Temporary

**How to use**:
1. Click "Employment Type" header to expand
2. Select the type(s) you're looking for

**Levels available**:

### 3. Salary Range Slider

**What it does**: Filter jobs by salary range

**How to use**:

**Method 1: Using the slider**
1. Look at the visual salary slider
2. Drag the left handle to set minimum salary
3. Drag the right handle to set maximum salary
4. Current range displays at the top
5. Results update as you adjust

**Method 2: Manual entry**
1. Click in the "Minimum" field
2. Enter desired minimum salary
3. Click in the "Maximum" field
4. Enter desired maximum salary
5. Results filter automatically

**Example**:
- Minimum: 50000
- Maximum: 150000
- Shows all jobs paying between 50K-150K

**Note**: The slider shows the actual salary range available in the database (e.g., "Min: 30,000 | Max: 250,000")

### 5. Saved Filters (For Registered Users)

**What it does**: Save your favorite search criteria for quick reuse

#### Saving a Filter
1. Set up filters (position, location, salary, etc.)
2. Click "Save This Search" in the Saved Filters panel
3. Enter a name (e.g., "Harare Pharmacist")
4. Enter optional description
5. Filter is saved automatically

#### Using Saved Filters
1. Look at "My Saved Filters" section
2. Click "Apply" on any saved filter
3. All filters auto-populate with saved values
4. Results update instantly

#### Managing Saved Filters
1. **Apply**: Click "Apply" button to use the filter
2. **Star**: Click star icon to set as default
   - Default filter loads when you visit the page
3. **Edit**: Click pencil icon to modify the filter
4. **Delete**: Click trash icon to remove the filter

#### Filter Information
Each saved filter shows:
- Filter name and description
- Tags: "3 positions", "2 locations", "Salary set"
- Usage stats: "Used 5 times ‚Ä¢ Last: Jan 15, 2024"

### 6. Active Filter Counter

**What it shows**: Number of filters currently applied

**Where to find it**:
- In the header (e.g., "Results: 45 found (3 filters)")
- Clear button shows "(4)" indicating 4 filters applied

**What counts as a filter**:
- Each search query
- Each selected position
- Each selected location
- Salary range (as one filter)
- Each selected employment type
- Selected experience level

### 7. Pagination

**When it appears**: When there are more than 20 results

**How to use**:
1. Review the current page results
2. Click "Next" to see more jobs
3. Click "Previous" to go back
4. Current page shown: "Page 2 of 5"
5. Limited to 20 results per page for performance

1. Check "Pharmacist" in Position filter

4. Later: Check saved filter periodically for new jobs

## Troubleshooting

### No Results Showing
**Problem**: Applied filters but getting "No jobs found"
**Solutions**:
1. Click "Clear All Filters" and try simpler search
2. Check salary range - might be too restrictive
3. Try different location or position
4. Verify filters are actually applied (count badge visible)

### Results Not Updating
**Problem**: Changed a filter but results stayed the same
**Solutions**:
1. Wait a moment - search processes on demand
2. Refresh the page
3. Check internet connection
4. Try a different filter first

### Can't Find Saved Filter
**Problem**: Saved a filter but don't see it
**Solutions**:
1. Are you logged in? Saved filters require login
2. Filter name might be different than expected
3. Try scrolling "My Saved Filters" section
4. It may have been deleted - save it again

### Salary Slider Not Working
**Problem**: Dragging slider has no effect
**Solutions**:
1. Refresh the page
2. Ensure minimum ‚â§ maximum
3. Check that salary values are in valid range
4. Try manual input fields instead
