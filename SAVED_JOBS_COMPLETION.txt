                            QUICK START GUIDE
================================================================================

1. INSTALLATION:
   npm install node-cron

2. CONFIGURATION (.env):
   EMAIL_HOST=your-smtp-server
   EMAIL_PORT=587
   EMAIL_USER=your-email
   EMAIL_PASS=your-password
   EMAIL_FROM=noreply@zimpharmhub.com
   FRONTEND_URL=https://zimpharmhub.com

3. TEST AN ENDPOINT:
   curl http://localhost:5000/api/saved-jobs \
     -H "user-id: your-user-id"

4. INITIALIZE REMINDERS (in server startup):
   const { initializeReminderScheduler } = require('./utils/reminderScheduler');
   initializeReminderScheduler();

================================================================================
                          API QUICK REFERENCE
================================================================================

SAVE A JOB:
  POST /api/saved-jobs
  Headers: user-id: {userId}
  Body: {jobId, notes?, emailReminderEnabled?, reminderFrequency?}

GET ALL SAVED JOBS:
  GET /api/saved-jobs
  Headers: user-id: {userId}

CHECK IF SAVED:
  GET /api/saved-jobs/check/{jobId}
  Headers: user-id: {userId}

UPDATE REMINDER SETTINGS:
  PUT /api/saved-jobs/{savedJobId}
  Headers: user-id: {userId}
  Body: {emailReminderEnabled?, reminderFrequency?}

REMOVE SAVED JOB:
  DELETE /api/saved-jobs/{savedJobId}
  Headers: user-id: {userId}

GET DASHBOARD:
  GET /api/dashboard
  Headers: user-id: {userId}

See SAVED_JOBS_ENDPOINTS.md for complete endpoint documentation.
